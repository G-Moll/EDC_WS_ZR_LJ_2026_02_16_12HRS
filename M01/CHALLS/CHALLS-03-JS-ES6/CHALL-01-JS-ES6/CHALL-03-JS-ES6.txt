CHALLENGE 03: HISTORIAL DE COMPRAS DE USUARIOS
Problema General:
Una plataforma e-commerce quiere personalizar la experiencia de sus usuarios analizando su historial de compras.

Tareas a Resolver:
    - Para un usuario dado, listar todos los nombres de los productos que ha comprado.
    - Calcular el gasto total de un usuario específico.
    - Determinar si un usuario específico ha comprado un producto en particular (por ejemplo, 'laptop').
    - Crear un perfil de usuario que muestre su nombre y la cantidad total de productos comprados.

Datos de Entrada:
const users = [
    { id: 1, name: 'Ana', purchases: [ 'laptop', 'mouse', 'headset' ] },
    { id: 2, name: 'Carlos', purchases: [ 'keyboard', 'monitor' ] },
    { id: 3, name: 'María', purchases: [ 'laptop', 'monitor', 'mouse', 'usb' ] }
];
const products = [
    { id: 1, name: 'laptop', price: 800 },
    { id: 2, name: 'mouse', price: 25 },
    { id: 3, name: 'headset', price: 50 },
    { id: 4, name: 'keyboard', price: 60 },
    { id: 5, name: 'monitor', price: 200 },
    { id: 6, name: 'usb', price: 15 }
];
