CHALLENGE 02: EVALUACIÓN DE DESEMPEÑO DE EMPLEADOS
Problema General:
El departamento de RRHH debe evaluar el desempeño de los empleados para determinar bonificaciones.

Tareas a Resolver:
    - Calcular el puntaje promedio de cada empleado basado en sus evaluaciones.
    - Filtrar a los empleados que tengan un puntaje promedio mayor o igual a 85 (empleados con bono).
    - Filtrar a los empleados que tengan al menos una calificación menor a 70 (necesitan mejorar).
    - Generar un mensaje para cada empleado que incluya su nombre y si califica para el bono ("Sí" o "No").

Datos de Entrada:
const employees = [
  { name: 'Elena', scores: [ 90, 88, 75, 92 ] },
  { name: 'Luis', scores: [ 68, 72, 70, 65 ] },
  { name: 'Sofía', scores: [ 95, 94, 98, 91 ] },
  { name: 'Miguel', scores: [ 78, 82, 80, 79 ] }
];
